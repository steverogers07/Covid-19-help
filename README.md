# Covid-19-help
A Website containing all the information regarding Covid-19. Check the website

# Tech Stack
NodeJS
Express

MongoDB

Deployed on Heroku.

# Build and run
1.Clone Repo
    git clone https://github.com/shivamgupta1999/Covid-19Help.git
2.To install all the dependencies, enter command:
    npm install
3.To run:
    node app.js
